Cara menghitung Anova dengan SPSS

- Masukan data dalam Data View.
- Urutkan dari ada ke bawah.
  Contoh : var var
	 	1   2
		1   0
		1   4
		1   7
		2   8
		2   4
		2   5
		2   9
		3   3
		3   8
		3   1
		3   4
- Pindah ke Variabel View.
- Ubah name 1 = Prosedur, name 2 = Data.
- Ubah Values dalam tabel 1, isi dengan(1 = A, 2 = B, 3 = C).
- Masuk ke menu Analyze -> Compare Means -> One-Way Anova.
- Pindah prosedur ke Factor, dan pindah Data ke Dependent List.
