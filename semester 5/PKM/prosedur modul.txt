prosedur modul:
- install sublime text
- buat akun github
	- gunakan email pribadi
- install git (centang open git bash)
	- setting akun:
		- git config --global user.email "email-github@gmail.com"
		- git config --global user.name "username-github"
		- close panel git
- buat repository pada github dengan nama: username-github.github.io, Add a README file di centang
- clone repository ke Desktop
	- buka repository username-github.github.io pada github
	- klik button code
	- salin https
	- di Desktop klik kanan git bash here
	- ketik git clone paste url yang telah disalin
	- tekan enter
	- jika sudah ter-clone, close panel git-nya
- buka sublime text 
	- new window
	- drag folder username-github.github.io ke sublime text
- mulai project
	- buka folder username-github.github.io 
	- buat file index.html
		- ganti favicon
	- buat file style.css
		- link css
	- koding
	- embed google maps
- klik kanan pada folder username-github.github.io
- ketik git add .
- ketik git commit -m "isi pesan"
- ketik git push
- login open with browser
	- close tab browser
- preview project portofolio dengan cara ketik url: username-github.github.io

- mengupdate project portofolio jika ada perubahan:
	- ubah file yang diinginkan
	- buka folder username-github.github.io klik kanan git bash here
	- ketik git add .
	- ketik git commit -m "isi pesan perubahan"
	- ketik git push
- buka github.com/username-github/username-github.github.io
	- di bawah tombol code, di sebelah kiri commit terdapat indicator
		- jika berwarna orange berarti sedang memproses website
		- jika sudah checklist berwarna hijau berarti website siap digunakan
- preview kembali
